Expand description

Date and time representation.

Fields§

§year: u16

Year. Valid range: 1900..=9999.

§month: u8

Month. Valid range: 1..=12.

§day: u8

Day of the month. Valid range: 1..=31.

§hour: u8

Hour. Valid range: 0..=23.

§minute: u8

Minute. Valid range: 0..=59.

§second: u8

Second. Valid range: 0..=59.

§pad1: u8

Unused padding.

§nanosecond: u32

Nanosececond. Valid range: 0..=999_999_999.

§time_zone: i16

Offset in minutes from UTC. Valid range: -1440..=1440, or Time::UNSPECIFIED_TIMEZONE.

§daylight: Daylight

Daylight savings time information.

§pad2: u8

Unused padding.
